# 🧾 MASTER FORMULA REFERENCE SHEET

> **Every formula from the entire course in one place.**

```
PROBABILITY
P(A) = |A|/|S|                              Classical probability
P(Aᶜ) = 1 - P(A)                            Complement rule
P(A∪B) = P(A) + P(B) - P(A∩B)               Addition rule (general)
P(A|B) = P(A∩B)/P(B)                        Conditional probability
P(A∩B) = P(A|B) × P(B)                      Multiplication rule
```
